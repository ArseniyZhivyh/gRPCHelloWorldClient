```markdown
# HelloWorld gRPC Client Application

## Обзор

Это простейшее клиентское приложение, которое использует gRPC для вызова удаленного сервера и отправки приветственного сообщения. 

## gRPC Сервер

Клиентское приложение предназначено для взаимодействия с сервером HelloWorld, расположенным по адресу:

[https://github.com/ArseniyZhivyh/gRPChelloWorld/tree/master]

Перед запуском клиентского приложения убедитесь, что сервер запущен и доступен.

## Запуск Клиента

Клонируйте репозиторий и запустите App.main()

После запуска приложение отправит запрос к серверу и, если соединение успешно, вы получите ответное приветственное сообщение.

```
